HTML Cheatsheet

Element Function Syntax
Unordered List Creates a bulletpoint list < ul >
...
< /ul>
Ordered List Creates a numbered list < ol >
...
< /ol>
List Creates list items within an ordered or unordered list ...< li >list item< /li>...
Video Inserts a video to the webpage and a comment if video fails < video src= "url" > Video not supported < /video>
Image Inserts an image to the webpage, from a source < img src= "url" > OR < img src = "filepath" >
Alt Attribute Offers description if image cannot be displayed < img src= "" alt= "image description" >
Anchor Element Creates hyperlink to section of the page, or to a completely different website.
Can be wrapped around element to make it a hyperlink.		 < a href= "url" >element to wrap< /a>
Line Break Creates a break, puts following text on the next line. "Text on first line" < br >
"text on next line"
